# 🎮 Tic-Tac-Toe Game

A fun and simple **Tic-Tac-Toe** game built with **HTML, CSS, and JavaScript**.  
Play against a friend or challenge the computer (AI that sometimes makes mistakes 😉).

---

## 🚀 Features
- 2 Player Mode  
- Play vs Computer/AI  
- Reset button to start a new game  
- Responsive design  

---
